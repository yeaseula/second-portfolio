## React · Typescript

### 학습 내용

- • 컴포넌트 단위로 UI를 분리, 상태와 렌더링 흐름을 고려한 설계
- • 기능과 UI의 책임을 분리하는 로직 설계
- • TypeScript의 인터페이스와 제네릭을 사용해 컴포넌트 props와 상태 타입을 명확히 정의하며, 실제 프로젝트에 적용하며 학습 중

### 관련 프로젝트

>**fish market | 소셜미디어 서비스**

관상어에 관심 있는 유저를 대상으로 한 소셜미디어 서비스입니다.
