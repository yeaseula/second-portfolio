## 자바스크립트 • 인터랙션 구현

### 학습 내용

- • 클래스 기반 로직으로 상태와 동작을 분리
- • DOM 이벤트와 애니메이션 흐름을 직접 제어하며 사용자 인터랙션 처리 경험
- • Three.js를 활용해 기본적인 3D 오브젝트 생성과 회전, 간단한 인터랙션 효과 구현
